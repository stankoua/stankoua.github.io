# Post Hello Word 

Voici une triviale application Hello Word en `Scala`:

```scala
object Main extends App {
  def main(args: Array[String]): Unit =
    println("Hello World")
}
```

Et en `Haskell`:

```haskell
module Main where

main :: IO ()
main = putStrLn "Hello World"
```
